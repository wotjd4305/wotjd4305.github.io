### 서울시의 구의 어떤 동에 상가를 시작하는것이 좋을까에 대한 추천 시스템.

**기간** : 2020.08 ~ 2020.09

```java
# 목표
서울시의 구의 어떤 동에 상가를 시작하는것이 좋을까에 대한 추천 시스템.

# 데이터 수집 방법 및 전처리
서울 열린 데이터 광장이나 부동산사이트의 전세가나 매매가를 크롤링
크롤링은 파이썬 퓨티불 숲 사용.
구글 지오 코딩을 통해서, 해당 데이터의 좌표, 구, 동을 구하였음.

# 수집 데이터
[1] 부동산 데이터(전세가율)는 2020년 5월까지는 긁을수 있어요
[2] 회사분포 데이터 [찾음]
[3] 음식점 분포 데이터
[5] 인구관련(유동|거주) [찾음]
[6] 환승지점 분포 데이터(교통이 좋다) [찾음] 
[7] 상권영역

# 데이터 변환
1) 판다스를 이용해서 CSV파일을 데이터 프레임으로 바꾸었고
2) 각 데이터들의 단위가 다르기 때문에, 정규화(0~1사이의값)
 -> 최소-최대 정규화 사용, ( x - min ) / (max - min)

# 추천
1) 협업 필터링 사용
 상가 종류(커피, 밥)에 따른 추천 점수(상가, 회사, 역 등)를 정하고, 협업 필터링 돌려서 가장 가까운 00구 00동 추천.
2) 선택한 상가(카페, 음식점)에 가중치를 줬음.
```
